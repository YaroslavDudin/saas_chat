# 🚀 SaaS Chat: Платформа Интеллектуальной Автоматизации

Комплексное решение для создания, управления и развертывания чат-виджетов с ИИ для автоматизации общения с клиентами и генерации лидов.

---

## 🌟 Для Клиентов и Пользователей

Добро пожаловать в будущее взаимодействия с клиентами! SaaS Chat позволяет создавать интерактивные сценарии, которые общаются с вашими пользователями 24/7.

### Ключевые возможности
- **Визуальный конструктор**: Создавайте сложные цепочки диалогов с помощью простого интерфейса (drag-and-drop).
- **Разнообразие блоков**: Используйте текстовые сообщения, вопросы, выбор кнопками и формы сбора контактов.
- **Мгновенная установка**: Скопируйте одну строку кода и вставьте её на свой сайт.
- **Управление лидами**: Просматривайте все данные, собранные ботами, в удобном личном кабинете.
- **Брендинг**: Настраивайте цвета виджета под стиль вашего бренда.

### 🛠 Руководство по установке
Запустить бота на сайте можно за 3 простых шага:

1.  **Создайте бота**: Зайдите в панель управления и нажмите "Создать бота".
2.  **Настройте сценарий**: В конструкторе добавьте нужные блоки и связи. Нажмите "Опубликовать".
3.  **Установите код**: Нажмите на иконку "Получить код" на карточке бота и вставьте скрипт перед закрывающим тегом `</body>` на вашем сайте.

```html
<!-- Пример кода установки -->
<script
    id="saas-chat-script"
    data-widget-id="ВАШ-ID-БОТА"
    src="https://your-domain.com/dist/widget.js">
</script>
```

## 🛠 Для Разработчиков

SaaS Chat построен на базе современной масштабируемой архитектуры, ориентированной на высокую производительность.

### 🏗 Обзор архитектуры
Проект представляет собой монорепозиторий, разделенный на три основные части:
1.  **`/server`**: Django REST Framework (Python). Обрабатывает API, базу данных (PostgreSQL) и логику переходов по сценарию.
2.  **`/dashboard`**: React (Vite + TypeScript + Tailwind). Панель управления и редактор узлов на базе `@xyflow/react`.
3.  **`/client`**: Легковесный виджет на React. Компилируется в один файл `widget.js` со встроенными стилями.

### 🚀 Основные команды

#### Запуск проекта (Docker)
```bash
# Сборка и запуск всех контейнеров
docker-compose up --build

# Запуск в фоновом режиме
docker-compose up -d

# Остановка и удаление контейнеров
docker-compose down
```

#### Разработка клиентского виджета (`/client`)
```bash
# Установка зависимостей
cd client && npm install

# Запуск в режиме разработки (HMR)
npm run dev

# Сборка виджета в один файл (dist/widget.js)
npm run build
```

#### Разработка панели управления (`/dashboard`)
```bash
# Запуск в режиме разработки
cd dashboard && npm run dev
```

#### Работа с бэкендом (`/server`)
```bash
# Применение миграций (внутри контейнера)
docker exec -it saas_chat_web python manage.py migrate

# Создание суперпользователя
docker exec -it saas_chat_web python manage.py createsuperuser
```

### 💻 Технические особенности

#### Сервер (Django)
- **Auth**: Токены для авторизации в панели управления.
- **Модели**: `Bot`, `ScenarioNode` (связаны через UUID) и `Lead` (история чата + структурированные данные).
- **Логика**: Эндпоинт `respond` динамически вычисляет следующий шаг на основе текущего состояния и ввода пользователя.

#### Панель управления (React + React Flow)
- Использует `@xyflow/react` для визуального редактирования графов.
- Кастомные компоненты узлов для каждого типа шага.
- Прямая интеграция с Django API через Axios.

#### Виджет клиента (Tailwind 4 + Vite)
- **CSS-in-JS**: Стили внедряются автоматически через `vite-plugin-css-injected-by-js`.
- **Изоляция**: Все CSS-классы используют префикс `twbot:`, чтобы избежать конфликтов со стилями основного сайта.
- **Сборка**: `npm run build` создает переносимый файл `widget.js`.
